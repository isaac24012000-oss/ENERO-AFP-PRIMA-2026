# AFP PRIMA ENERO 2026

Este repositorio contiene el dashboard de gestión y análisis para AFP PRIMA WORLDTEL Enero 2026, desarrollado en Streamlit y Python.

## ¿Qué incluye?
- Visualización de KPIs y métricas clave
- Tablas resumen por campaña, asesor y prioridad
- Análisis estratégico por nivel de riesgo (con Prioridad 15 como la más actual)
- Detalle de casos críticos (Prioridad 15 + Contacto Directo + Sin Pago)
- Gráficos interactivos y tablas estilizadas

## Requisitos
- Python 3.11+
- Streamlit
- Pandas
- Numpy
- Matplotlib
- Openpyxl

## Ejecución
1. Instala las dependencias:
   ```bash
   pip install streamlit pandas numpy matplotlib openpyxl
   ```
2. Ejecuta el dashboard:
   ```bash
   streamlit run dashboardEnero2026.py
   ```

## Estructura
- `dashboardEnero2026.py`: Código principal del dashboard
- `DATA TOTAL WORLDTEL ENERO 2026.xlsx`: Archivo de datos (no incluido por privacidad)

## Notas
- El archivo de datos debe estar en la ruta indicada en el código.
- Personaliza el dashboard según tus necesidades.
- Sistema de prioridades: La Prioridad 15 representa el período más actual (Enero 2026).
